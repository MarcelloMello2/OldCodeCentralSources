Show Project Path in IDE title
-------------------------------


Author: Wayne Schou

- The path of the currently loaded project is shown in the title bar (replacing the words "Delphi 6").
- The caption is updated everytime a file is opened or closed.
- Has no options or other visual elements.
- Useful when you have several versions of the same project.


- Installation instructions:
Install just like any other design-time package.
