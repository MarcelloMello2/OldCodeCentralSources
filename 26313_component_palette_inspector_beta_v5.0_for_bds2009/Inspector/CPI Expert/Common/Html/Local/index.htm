What's new in V50

Component Palette Inspector (CPI) is a Delphi expert that speeds users progress in Delphi applications development by providing various helpful services in the following categories: component palette, form designer, property inspector , debugging services  and IDE.

CPI V50 provides the following services for users:
    Components Palette Services

CPI IDE Explorer

All operations below are done through this expert
  1. Context Explorations of components
     
  2. GENERAL
    1. Helps users to explore the insides of Delphi IDE
    2. Explores all Delphi child forms
    3. Explore CPI itself
       
  3. TREES
    1. Provides an on-demand exploration tree of component with the followings nodes
      • "Owned", "Parented", "Properties", Special", and " hierarchy"

    2. Provides a full-tree structure of explored components with the following nodes
      • History, components, classes
      • Enumerations, actions, events
      • Types and icons.
  4. EXECUTION
    1. Execute actions in explored components
    2. Execute notify event methods
       
  5. SIMULATION
    1. Simulate explored menus by reconstructing menu from its elements
    2. Simulate popup menus
    3. Simulate sub menus attached to menu items.
       
  6. MODIFICATION
    1. Provides run-time properties inspector to modify properties
    2. Provides color property modifier
       
  7.  SOURCE CODE GENERATION
    1. Helps users to generate a source-code to explore Delphi IDE components;
    2. Generates GET property value source code
    3. Generates SET property value source code
    4. Generates RUN source code for methods
    5. Generates EXECUTE source code for methods and actions
    6. Generates EXAMPLES source code
    7. Replace
    8. Intercept
    9. Generates full units source code for accessing all properties of a component
       
  8. INSPECTING
    1. Provides list-view for inspecting properties
    2. Provides a tree-view for inspecting properties
    3. Provides list-view for inspecting events/methods
    4. Provides a tree-view for inspecting events/methods
    5. Provides a three basic hierarchies of component which are
      • Classes hierarchy
      • Parents hierarchy
      • Owners hierarchy
         
  9. CONTENT
    1. Lists contents of menus
    2. Lists content of list boxes, check boxes
    3. Lists content of image-lists images
    4. List contents of tool bar images
    5. Lists content of tree and list views

Feel Happy
For having an expert like CPI Expert

When comparing Delphi to other programming languages like C#, C++, VB, Java, ...etc. one finds that it is still one of the best programming language and many thanks for all the people in Borland. One of the most beautiful thing about Delphi, it offers an Open Tools API to allow programmers to extend the functionality of Delphi IDE . Without this feature, this work would be impossible.

CPI expert is a personal effort done by me during my spare time and it is not a company product at all and not owned by anybody except me. So, you may use its beta version as a freeware only. I will try my best to keep it updated and make it a professional expert that makes life easy while using Delphi even though my programming knowledge is limited

and acquired through self-learning ( I have a B.S and M.S degrees in Physics and a B.S. degree in Electrical Engineering).

CPI V50 expert reached 151,000 lines of source code.  Last compile NOV 22, 2008.  Irregardless of this hard work, Delphi deserves this sacrifice.
