For installation:

  • Either open AutoTodoInsert.Dpk and install the package, or
  • Add uAutoTodo.pas and uIdeUtils.pas to an existing design-only package and rebuild that package.

The Auto-insert command will appear in the edit menu.

This software has been tested for Delphi 6 only.
