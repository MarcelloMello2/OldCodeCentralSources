CPI Array and List Palette inside BDS tools form

Now inside BDS ToolForm CPI presents an Array Palette to display components by their bitmaps in an array of rows and columns as shown in figures below.  Now users have the choice to switch between two views: List View and Array View as shown below.  In list view you can hide or show 'T' prefix for component classes.  Hiding 'T' simplifies locating components visually.  In all views, you search within displayed components or within all installed components.  "Search as you type" is also provided for array palette, list palette and BDS original palette.
