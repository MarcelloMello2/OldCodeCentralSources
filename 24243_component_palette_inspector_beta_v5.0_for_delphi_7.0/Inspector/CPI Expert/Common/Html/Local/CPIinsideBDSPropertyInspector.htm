CPI inside BDS Property Inspector
with more controls

 

CPI in V40 enhances BDS property inspector by adding the following pages

  1. BDS Sheet
    Contains BDS properties list inspector
     

  2. List sheet
    Lists all components in active form editor organized by types.  Each group is has component class name and  bitmap as shown below.

     

  3. Search sheet
    Provides a quick search for a component by its name in form designer.  Found results are organized by types.
     

  4. Selection History sheet
    Provides history for last 50 selected components organized by types.

Addition features

  1. All form designer components are listed and organized by their class names. 

  2. Provides a quick search sheet.  For advanced search schemes use CPI.

  3. History of selected components organized by their class names

  4. Popup Menu of all sheets allows users to

    1. Select components in form designer automatically

    2. Select components in different controls (important)

    3. Edit all components with component editors

 

If you place a component in form editor and it becomes under other controls so it is invisible, use "List Sheet"  to locate component.
