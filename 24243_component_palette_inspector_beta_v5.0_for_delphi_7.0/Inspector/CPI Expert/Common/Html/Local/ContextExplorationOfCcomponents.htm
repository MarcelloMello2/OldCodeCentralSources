Context exploration of components

CPI V40 enables users to explore IDE components on the fly.  Just right click any component, and select from its popup menu 'Properties' as shown below.

Figure below shows BDS2006 options dialog. 

Figure below shows CPI exploration to red bounded component in above figure.

This explorer can generate source code to access explored component properties.  Users have to be careful when generating source code for explored components in dialogs.  Returned reference to components may becomes invalid after dialog shuts down because it destroys all its components and hence references become invalid.
