
Remote Assistant
================

Remote Assistant application, included in this package is the 
administrator-side software. User-side (host) software RemoteHost
is freeware and is is available on our web site at:
http://contextsoft.com/products/remassist/downloads.jsp

Please, refer to the Remote Assisstant help file on how to
customize and localize RemoteHost.exe and how to invoke 
it from your application. Demos and Delphi code for this can 
be found in this installation package and help file.
